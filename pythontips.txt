# Dictionary 

To create a dictionary 

empty_dict = {}

empty_dict = dict()

dict = {'key': 'value'}

---> to update single value

dict['key'] = value

----> to update multiple values 

dict.update({'multiple_keys': 'multiple_values'})

# Sets

Sets are usually used to remove duplicate data. 

empty_set = {} 	----> This is the wrong way to create empty set. This actually create an empty dictionary.

empty_set = set()    ----> The right way.

We can use methods like:

intersection()
difference()
union()

to determine the value on the sets.
